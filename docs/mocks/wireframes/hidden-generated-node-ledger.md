# Hidden Generated-Node Ledger

This ledger tracks implemented pages that still hide page-local generated `.pen` nodes at runtime. The goal is to keep those suppressions visible to reviewers until each page no longer needs the static generated nodes, or until the runtime surface can take ownership without relying on hidden artboard geometry.

Shared-shell suppression is not tracked here. The shared shell intentionally hides duplicate nav, search placeholder, and role-filtered sidebar nodes while the runtime shell overlays live controls. This ledger is only for page-local generated nodes on issue #11's assigned pages.
